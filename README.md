# Concesionario
Sistema de Información basado en la administración de un concesionario.
Somos una empresa llamada ALSI System dedica a la comercialización y exportación de productos tecnológicos de alta calidad para la fabricación de computadores, la empresa ha tenido un gran crecimiento en los últimos años por lo que decidió implementar un departamento de TI el cual apoye a la organización creando estrategias para cumplir con el objetivo de esta, Se plantea un proyecto en el cual se pueda gestionar un concesionario de tal forma que se conozca información de comprador e información relacionada a la compra.
